# Ben project

This is the project repository for **Ben**, a greenfield project (which was built from a template), that functions as a personal assistant chatbot.

## Setting up with the `.jar` file

Prerequisites: JDK 17

1. Download the `.jar` file from the latest release.
2. Navigate to the directory that contains the `.jar` file in the command line.
3. Run `java -jar [Name of the .jar file].jar`
